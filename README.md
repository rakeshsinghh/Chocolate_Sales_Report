# 🍫 Global Chocolate Sales & Market Insights Report

An end-to-end Power BI analytics project built to analyze global chocolate sales performance across 2023–2024 using interactive dashboards, KPI tracking, and business intelligence techniques.

## 📊 Project Highlights

* 💰 **25.49M Total Revenue** analyzed across global markets
* 📈 **10.19M Total Profit** with **40% Profit Margin**
* 🛒 **1M+ Orders** and **3M Products Sold**
* 👥 Insights generated from **50K Customers**
* 🌍 Country-wise sales analysis across Canada, UK, USA, France, Australia, and Germany
* 🍫 Brand performance comparison including Ferrero, Cadbury, Lindt, Hershey, Mars, and Godiva
* 🔄 **79.88% Repeat Customer Rate** analysis
* 📉 Discount impact analysis on revenue and profitability
* 📅 Monthly sales trend and MoM growth analysis
* 🧠 Star Schema data model designed for optimized BI reporting

## 🛠 Tools & Technologies

* Business Intelligence
